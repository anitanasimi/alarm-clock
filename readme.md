# Clock and Alarm Application

This is a simple web application that displays a clock with the current time and allows you to set an alarm.

## Features

- **Clock Display**: The clock displays the current time in hours, minutes, seconds, and session (AM/PM).
- **Alarm Functionality**: Set an alarm by selecting the hour, minute, and session (AM/PM) from the dropdown menus and clicking the "Set Alarm" button. The alarm will ring when the set time matches the current time.

## How to Use

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser.
3. The clock will start displaying the current time and update every second.
4. Use the dropdown menus to set the alarm time and click the "Set Alarm" button.
5. When the alarm time matches the current time, a ringtone will play.
